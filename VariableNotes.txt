// In a byte the max values are -128 to 127
// The first 0 or 1 in a byte is a sign byte
// and each byte is 2 to the power of its position
// so it is + or -, 1x2^6, 0x2^5 all the way to 1x2^0
// The 1 or 0 decides whether or not it is a 0 or 1 in the multiplication position
